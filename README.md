Hospital Website
📌 Overview
This project is a static hospital website built using HTML, CSS, and JavaScript.
It was created as part of my university coursework to practice front‑end development, UI design, and basic website structure.
The website includes essential hospital sections such as services, departments, doctors, appointments, and contact information.

✨ Features
- Clean and modern hospital-themed UI
- Responsive layout for mobile and desktop
- Sections for:
- Home
- About the Hospital
- Departments
- Doctors
- Services
- Contact / Appointment Form
- Smooth scrolling and simple JavaScript interactions
- Static website — no backend required

🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
